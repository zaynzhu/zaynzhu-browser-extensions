# Browser Search Extensions

右键选中文字，快速在影视网站搜索。

## 扩展列表

### HDHive Search

在任意网页选中文字后右键，点击「在 HDHive 搜索」即可在新标签页打开 HDHive 搜索结果。

- 网站：https://hdhive.com
- 搜索类型：聚合搜索（电影 + 剧集）

### 豆瓣搜索

在任意网页选中文字后右键，点击「在豆瓣搜索」即可在新标签页打开豆瓣搜索结果。

- 网站：https://search.douban.com

## 安装方法

1. 下载对应扩展目录
2. Chrome 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择扩展目录
